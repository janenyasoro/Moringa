# Adventures in Fire Island 🌊

A beautiful, responsive multi-page website showcasing the natural beauty of Fire Island National Seashore. Built with pure HTML/CSS using Flexbox and Grid layouts.

## 🏖️ Live Demo
[View the website](https://yourusername.github.io/fire-island-website)

## 📋 Features

- **3 Complete Pages**: Home, Gallery, and Contact pages
- **Responsive Design**: Fully responsive layout working on all devices
- **Modern CSS**: Built with CSS Grid and Flexbox (no frameworks)
- **Complementary Color Scheme**: Earthy greens and warm yellows
- **Interactive Elements**: Gallery filtering, contact form, hover effects
- **Clean Navigation**: Sticky navbar with active page highlighting

## 🎨 Color Palette

- Primary: #2C5F2D (Forest Green)
- Secondary: #97BC62 (Grass Green)  
- Accent: #FFE77A (Warm Yellow)
